TopDownload Extension Privacy Policy

Effective date: April 22, 2026

This privacy policy applies to the browser extension TopDownload Link Downloader ("the Extension").

1. Data Collection

The Extension does not collect, store, or sell personal data such as name, email, account credentials, or browsing history on remote servers.

2. How the Extension Works

The Extension helps users send downloadable links from the browser to the local TopDownload desktop app via the custom protocol topdownload://.

3. Permissions Usage

  • contextMenus: Adds the right-click menu item for links.
  • tabs: Opens the topdownload:// link in a browser tab to trigger the local app.
  • downloads: Detects browser download creation and optionally redirects matched downloads to TopDownload.
  • <all_urls> host permission: Allows detection of downloadable links on visited pages.

4. Data Sharing

The Extension does not transfer user data to third-party analytics, advertising, or cloud processing services.

5. Data Storage

The Extension does not maintain a remote database. Any handling occurs locally in the browser and the local TopDownload app environment.

6. Security

The Extension follows the browser extension security model and only requests permissions necessary for download handoff functionality.

7. Changes to This Policy

We may update this policy from time to time. The updated version will be posted at the same URL with an updated effective date.

TopDownload 扩展隐私政策

生效日期：2026 年 4 月 22 日

本隐私政策适用于浏览器扩展 TopDownload 链接下载（以下简称“本扩展”）。

1. 数据收集

本扩展不会收集、存储或出售用户个人身份信息（如姓名、邮箱、账号密码）或上传完整浏览历史到远程服务器。

2. 功能说明

本扩展用于将网页中的下载链接通过 topdownload:// 协议发送给本地 TopDownload 桌面应用进行下载处理。

3. 权限用途

  • contextMenus：在网页链接右键菜单中增加“使用 TopDownload 下载”。
  • tabs：在新标签页打开 topdownload:// 链接以唤起本地应用。
  • downloads：检测浏览器下载创建事件，并按规则转交给 TopDownload。
  • <all_urls>：用于识别网页中的可下载链接。

4. 数据共享

本扩展不会将用户数据提供给广告、统计分析或其他第三方服务。

5. 数据存储

本扩展不建立远程数据库。相关处理仅在用户本地浏览器与本地 TopDownload 应用环境中进行。

6. 安全措施

本扩展遵循浏览器扩展安全模型，仅申请实现下载接管所必需的权限。

7. 政策更新

如有更新，我们会在本页面发布新版本并更新生效日期。
